IT Camp



On 12th February 2015 MsSarita, and Ms. Poonam Assistant Professors from Dronacharya college of Engineeirng, Gurgaon attended “Implementing a mobile -first enterprise device strategy IT camps” by Microsoftat Radisson Blu Hotel Dwarka, New Delhi.

Ms.Pracheta Budhwar (Technology Evangelist at Microsoft Corporation India Pvt. Ltd) & Mr.Ninadwere were the speaker of the Microsoft IT camp.

The Main Agenda for this IT camp was Introduction to Microsoft Azure Directory & implementing MFA for Azure Accounts. Azure Active Directory is a service that provides identity and access management capabilities in the cloud. Application developers can integrate their applications with Azure AD to provide single sign-on functionality for their users. This enables enterprise applications to be hosted in the cloud and to easily authenticate users with corporate credentials. It also enables software as a service (SaaS) providers to make authentication easier for users in Azure AD organizations when authenticating to their services. Multi-factor or two-factor authentication (MFA) is a method of authentication that requires the use of more than one verification method and adds a critical second layer of security to user sign-ins and transactions.

In Lab Session topic with how to integrate Saas Apps with azure Ad was implemented.Session ended with the Q & A round.

Microsoft IT camp, Delhi provided an opportunity to get familiar with app Protection & Device Management with Intune. The event provided information on the Azure service that can be used for identity and access management capabilities in the cloud.
