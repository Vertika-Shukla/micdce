Microsoft Cloud OS Roadshow



On 22nd February 2014, Ms. Sarita, Assistant Professor from Dronacharya College of Engineering, Gurgaon, CSE Department, attended a one- day Microsoft Session entitled as “Microsoft Cloud OS Roadshow” held at Microsoft Cooperation Pvt. Ltd, Gurgaon. Event was organized by Mr Tanmay Kapoor (Community Support lead at Microsoft).

The Cloud OS Roadshow was a full day event hosted by Microsoft MVPs and Community members which lets the participants learn about Windows Azure and its features. Topics covered in this event were: Breakthrough Data Performance Platform, Virtualization, Building Modern Apps and Windows Azure, Introduction to Cloud Computing and Windows Azure Overview, Windows Azure Compute – IaaS, Websites and Cloud Services, Windows Azure Storage – SQL Database, Blob Storage, Table Storage.

The Main Aim of attending this event is to take a step ahead to get in touch with the real time IT environment, Interaction with MVP’s and other IT cooperate people such as Accenture and Cleaver clues to know how exactly the processing in a business takes place. Students from CSE Department Mr.Bibhuti Bhushan (CSE-3rd Year), Mr. Puneet Bhalla(CSE-3rd Year) Mr.Sahil (CSE-2nd Year) and Mr. Vikas Verma (CSE-2nd Year) also attended this Microsoft Session.

Microsoft’s Most Valuable Professional (MVP’s) Mr.Gaurav Mantri, Mr. Sharabpreet Singh, Mr. Shaantanu and Mr.Dhanjay Kumar took control of the Session.

First session:- The very first session of the event was handled by Mr Gaurav Mantri (Founder, Cynapta Software, and Windows Azure MVP) who helped audience to get close to the cloud world through Azure thing. He gave basic information of Cloud computing and its benefits. Speaker also put lights on Cloud as OS.

Second Session: The second session was headed by Mr Shaantanu Kaushik (Windows Expert-IT pro, Microsoft MVP) who took a step further by telling the concept of Hyper V and its latest versions

Third Session: This session was handled by Mr Sarabpreet Singh Anand (SQL Server Consultant) who told the new trend in Backend Designing that includes the concept of Databases.

Last session The last session of the day was handled by Mr Dhananjay Kumar (working at Telerik & Microsoft MVPs) who told about “Windows Azure mobile services as Backend service layer”. At the end of Session, Speakers gave “Microsoft goodies” to the audience on the basis of lucky draw. Event was very informatics and interactive. Faculty member & Students gained a lot of knowledge with this Cloud Roadshow.
