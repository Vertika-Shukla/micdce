2.TM System (Transport Management System):

TM System (Transport Management System) is an extremely useful and easy to understand software. It’s been vastly used by Companies which are engaged in the business of transportation. It’s a Dream System for any Transporter? This is one of the most appreciated and useful system for transporters.
Being a transporter either you are in the business of Full Truck Load, Part Load, Commission Agent Or Retail(Parchoon).We have four versions of TM System(Transport Management System) keeping in mind all the above mentioned types of transporters It is extremely an use full tool to enhance performance of organisation.

TM System Key Features:
(A) BOOKING MODULE:
• Consignment (Builty Booking As per Branch)
• Manifesto
• Despatch Challan
• Transshipment
• Comission Generator
• Stock Status(As Per Destination/Branch with in defined duration)
• Paid/To Pay/To Be Billed Booking Statement
• Stock Loss At Booking
• Service Tax As Per Booking Calculation

(B) DELIVERY MODULE:
• Arrival Challan
• Delivery Freight Memo
• Despatch/Delivery Register
• Integration with Rate Master
• Arrival Stock
• Delivery Stock
• Short/ Excess Register
• Commission Register
• More Than 30 Utility Reports
• Service Tax As Per Collection Calculation

(C) MAINTINANCE MODULE:
This Module is for maintaining the Truck’s Detail at the Transporter’s End. By using this module, the user will able in managing all type requirements. The user generally faces some problems at his end in managing the Truck’s Information, this module provide solution for such problems.
• Truck’s Detail > Information About > Truck’s Information
• Truck’s Detail > Information About > Trips
• Truck’s Detail > Information About > Baheti
• Truck’s Detail > Information About > Repairing
It has 2 – options like ‘During Trip’ & ‘Besides Trip’. In case of During Trip, user will Save the Repairing Detail for During Trip Cases only. In case of Besides of Trip, user will Save the Repairing Detail for Besides Trips Cases only.
• Truck’s Detail > Information About > Accidents
