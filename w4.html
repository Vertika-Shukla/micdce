WINDOWS AZURE



Microsoft Corporation is an American multinational corporation headquartered in Redmond, Washington, that develops, manufactures, licenses, supports and sells computer software, consumer electronics and personal computers and services. Its best known software products are the Microsoft windows line of operating systems, Microsft office office set, and Internet explorer, Web browser. Its flagship hardware products are Xbox game console and the Microsoft Surface series of tablets. It is the worlds largest software maker measured by revenues. It is also one of the worlds most valuable companies.

Microsoft has made its Windows Azure Web Sites and Mobile Service generally available in order to encourage developers to use its Azure cloud service to build scalable Web sites.

Tech Crunch is reporting that the service will allow users to develop a mobile backend for every device. This will then simplify user authentication, push notification, server and business logic. Microsoft will begin to offer its Mobile Services in three tiers: free, standard and premium. While all three tiers will be free through Aug. 1, the standard and premium tiers are metered by the number of API calls and backed by its service-level agreement (SLA). Mobile Services provides native SDKs for Windows Store, Windows Phone, Android, iOS and HTML5. Microsoft has taken the past few months to refocus more on mobility.

Yankee Group Principal Analyst Chris Marsh comments:
“The big guns are starting to show their hands in the growing mobile cloud marketplace. For example in recent months Rackspace and Salesforce both released a stack of mobile cloud services for developers, Google recently added, and now Microsoft has released on GA its Mobile Services based on Azure. All want to offer an integrated stack for mobile cloud applications to capitalize on both the consumer and increasingly the burgeoning enterprise mobile applications opportunities.”
